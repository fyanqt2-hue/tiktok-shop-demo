隐私政策

TikTok Shop 商家智能运营工具
更新日期：2026年6月7日 ｜ 生效日期：2026年6月7日
重要提示：本隐私政策说明我们如何收集、使用、存储和保护您的个人信息。访问或使用本工具即表示您同意我们按照本政策处理您的信息。

一、我们收集的信息

1.1 您主动提供的信息

  • 注册信息：当您注册使用本工具时，我们会收集您的 TikTok Shop 商家账号信息
  • API 授权信息：当您授权本工具连接 TikTok Shop API 时，我们会获取您的店铺访问权限令牌
  • 联系信息：如您通过邮件等方式联系我们，我们会收集您的联系方式

1.2 自动收集的信息

  • 使用数据：您使用本工具的功能记录、操作日志
  • 设备信息：您的浏览器类型、操作系统版本、设备型号等
  • 日志信息：访问日期时间、页面访问记录、API 调用记录

1.3 TikTok Shop API 获取的数据

基于您的授权，我们通过 TikTok Shop Open API 获取以下数据：

  • 商品信息（商品名称、价格、库存、SKU 等）
  • 订单信息（订单号、商品明细、物流状态等）
  • 数据分析（GMV、转化率、流量来源等）
  • 店铺数据（店铺概况、促销活动等）

二、信息的使用目的

我们仅将收集的信息用于以下目的：

  • 提供服务：为您提供商品管理、广告营销、数据采集、订单管理等功能
  • 数据同步：与 TikTok Shop 平台进行数据同步和交互
  • 用户体验优化：分析使用数据以改进产品功能和界面设计
  • 问题排查：诊断和修复技术故障，确保服务正常运行
  • 安全合规：验证身份、防范欺诈、维护服务安全、履行法律义务

三、信息的存储

3.1 存储地点

我们在中国境内的服务器上存储您的数据。涉及跨境数据传输的，我们将按照中国相关法律法规的要求采取必要的保护措施。

3.2 存储期限

  • 服务期间：您的数据和个人信息将一直被保存
  • 服务终止后：我们将在 30 天内删除或匿名化您的个人信息
  • 法律法规另有要求的，从其规定

3.3 安全措施

  • 数据传输：采用 HTTPS/TLS 加密传输
  • 数据存储：采用 AES-256 加密存储
  • 访问控制：严格的权限管理，限制内部人员访问
  • 安全审计：定期进行安全检查和漏洞修复

四、信息的共享与披露

4.1 共享范围

我们不会向任何第三方出售、出租或交易您的个人信息。仅在以下情况下共享信息：

  • TikTok Shop 平台：为提供服务所必需，通过 API 与 TikTok Shop 平台进行数据交互
  • 技术服务商：我们聘请的第三方服务商（如云服务器提供商、数据分析工具等），仅在其协助我们提供服务所必需的范围内共享
  • 法律要求：法律法规要求、法院命令或政府主管部门要求的情况下

4.2 共享保护

我们要求所有接收我们数据的第三方服务商遵守严格的数据保护标准，并与我们签署数据处理协议。

五、Cookie 与类似技术

我们使用 Cookie 和类似技术来：

  • 记住您的登录状态，免重复登录
  • 保存您的个性化设置和偏好
  • 分析服务使用情况，优化产品体验

您可以通过浏览器设置管理或删除 Cookie，但这可能影响部分功能的正常使用。

六、您的权利

  • 访问权：您有权查询我们持有的您的个人信息
  • 更正权：您有权要求更正不准确或不完整的个人信息
  • 删除权：您有权要求删除您的个人信息（法律要求保留的除外）
  • 撤回授权：您有权随时撤回对数据处理的授权
  • 注销账号：您有权注销账号并删除所有关联数据
  • 数据可携带权：您有权要求以通用格式获取您的数据

行使上述权利，请联系：privacy@example.com

七、未成年人保护

本工具主要面向成年商业用户。如果您未满 18 周岁，请在您的监护人陪同下使用本工具。我们不会故意收集未成年人的个人信息。如发现无意中收集了未成年人个人信息，我们将立即采取措施予以删除。

八、未成年人信息保护

我们非常重视对未成年人个人信息的保护。若您是 14 周岁以下（含 14 周岁）的未成年人，请在您的父母或监护人指导下使用我们的服务并填写我们的服务。若您发现未经父母或监护人同意情况下收集了您的个人信息，请立即联系我们，我们将尽快删除相关信息。

九、第三方链接

本工具可能包含指向第三方网站或服务的链接。我们不对第三方的隐私政策或 practices 负责。建议您阅读第三方的隐私政策。

十、政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面公布，重大更新将通过站内通知或邮件等方式告知您。请定期查看本页面以获取最新信息。

如您对本隐私政策有任何疑问、意见或建议，或需要行使您的数据权利，请通过以下方式联系我们：
